# Mount View Hotel Skardu - Frontend

## Quick Deploy to Vercel

### Method 1: Root Directory Setting (RECOMMENDED)

In Vercel Dashboard:
1. Set **Root Directory**: `frontend`
2. Framework will auto-detect as Vite
3. Click Deploy

### Method 2: Manual Settings

- **Build Command**: `yarn build`
- **Output Directory**: `dist`
- **Install Command**: `yarn install`
- **Node Version**: 20.x

## Local Development

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
```

Output: `dist/` folder
